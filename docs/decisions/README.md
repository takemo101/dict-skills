# 意思決定ログ (ADR)

重要な技術的判断・知見の詳細な経緯を記録する。

## フォーマット

```
NNN-問題名.md
```

## 記録一覧

| # | タイトル | 状態 |
|---|---------|------|
| [001](001-playwright-cli-session.md) | playwright-cli セッション問題 | 有効 |

## 記録すべき内容

- 外部依存の仕様変更対応
- 「これはダメだった」という知見
- 設計上の意図的な制限
- 環境依存の注意点

## テンプレート

```markdown
# NNN: タイトル (YYYY-MM-DD)

## 問題
...何が起きたか...

## 調査
...試したこと...

## 対応
...最終的な解決策...

## 影響
...コードや運用への影響...
```
